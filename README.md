# ObrasCerca — MVP

## 🧠 ¿Qué es?

ObrasCerca es una plataforma cívica que muestra las **obras públicas reales** sobre un mapa interactivo, con información oficial de tres fuentes del Estado peruano (MEF, Infobras y Contraloría).

El ciudadano ve cada obra georreferenciada y obtiene las respuestas que importan: **qué es, quién es responsable, cuánto cuesta, en qué estado está y cuándo termina** — siempre con cita a la fuente pública oficial.
